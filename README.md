# Deploy to GCP via Terraform 


#Local Use

## Pre-requisites

- Terraform
- Gcloud sdk

```bash
brew install terraform
brew install google-cloud-sdk
```

## Usage

```bash
terraform init
terraform apply
terraform destroy
```
